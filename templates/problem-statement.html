Problem Statement

Many skilled housewives have talents ranging from cooking and tailoring to tutoring and art. However, most lack a platform to promote their services or connect with potential clients. This project aims to bridge that gap by creating a web-based Enquiry Management System using Django, allowing housewives to receive and manage service requests from customers in an organized and professional manner.

Features

Core Features of the Enquiry Management System

Freelancer Profiles

Housewives can create profiles showcasing their services and availability to receive client enquiries.

Enquiry Management

Clients can submit enquiries and freelancers can view, manage, and respond to them via their dashboards.

Secure Login & Signup

Separate authentication for clients and freelancers ensures secure access and interaction.

Admin Dashboard

Admins can monitor user activity, approve freelancer accounts, and oversee the platform usage.

API Integration

The system is built using Django REST Framework to allow future expansion via APIs or mobile apps.

Mobile Responsive

The interface is fully responsive and works seamlessly across devices, allowing access on the go.
