Enquiry Management System for Freelancer Housewives

Empowering housewives by enabling them to manage their freelance work efficiently

Our system provides an easy way for freelancer housewives to manage their enquiries, helping them to stay organized and professional. The platform allows users to submit forms, view an admin dashboard, and use login/signup features for secure access.

  • Easy-to-use enquiry form submission
  • Admin dashboard for management
  • User authentication with login/signup

This project uses Django for backend development and Django REST Framework for API integration, ensuring scalability and performance.
